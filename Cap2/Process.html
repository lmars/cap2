- (Boolean) enable(capability)

Enable the given capability for this process.

Raises a RuntimeError if the process's pid is not the same as the current pid (you cannot enable capabilities for other processes, that's their job).

VALUE cap2_process_has_cap(VALUE self, VALUE set_sym, VALUE cap_sym) {
  cap_t cap_d;
  VALUE result;

  cap_d = cap2_process_caps(self);

  result = cap2_has_cap(cap_d, set_sym, cap_sym);

  cap_free(cap_d);

  return result;
}
